# Documentation

display

flex: The children of a flex container can be laid out in any direction, and can "flex" their sizes.
Either growing to fill unused space or shrinking to avoid overflowing the parent.

flex-direction: whether the items should be in a row or column

flex-wrap: whether or not items should be on one line or wrap into a second

justify-content: defines how the browser distributes space between and around content items (start,end,center,space-evenly,space-around)

overflow

align-items: aligns items with itself

text-align: sets horizontal alignment of the content inside a block element or table-cell box

transform: lets you rotate, scale, skew or translate an element

z-index: sets the z-order of a positioned elemnent and its descendents or flex items. Overlapping elements with a larger z-index coer those with a smaller one

transition: used to transition certain attributes - transition: margin-right 4s; for example

cursor: sets the mouse cursor to show when the mouse pointer is over an element
